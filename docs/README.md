## System Overview

BiasScope is an automated AI bias and fairness analysis tool that:

1. Takes only an AI model API URL as input
2. Automatically generates controlled synthetic test data
3. Evaluates the model for bias and fairness
4. Presents clear, visual insights through the frontend

No manual data input or model training required!
